# Paradise Nursery Shopping Application

## Project Overview
A full-featured React + TypeScript shopping cart application for **Paradise Nursery**, an online plant shop.

### Features
- Landing page with background image and "Get Started" button
- Browse plants by category (Houseplants, Succulents, Flowering Plants)
- Add to cart functionality with Redux
- Shopping cart management (add, remove, update quantity)
- Responsive design

Built with:
- React + TypeScript + Vite
- Redux Toolkit
- CSS for styling

## File Structure (Key Files)
- `src/App.tsx` - Main app with routing logic
- `src/components/AboutUs.tsx` - Company details
- `src/App.css` - Styles + background image
- `src/redux/cartSlice.ts` - Cart reducers
- `src/components/ProductList.tsx` - Product browsing
- `src/components/CartItem.tsx` - Cart items
